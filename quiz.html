💻 Part 2 — Programming Practice Ideas

These are the kinds of questions that could appear on the practical section. Practice writing them by hand on paper or in a plain text editor.

Idea 1: Array Reversal with Pointer Arithmetic

Write a function void reverse(int *arr, int size) that reverses the array in-place using only pointer arithmetic (no square-bracket indexing). Call it from main(), print the result with a loop.

Idea 2: Command-Line File Line Counter

Accept a filename as argv[1]. Open the file with fopen(); if it returns NULL, print an error and return 1. Use fgets() in a loop to count lines; print the total and close the file.

Idea 3: Student Grade Calculator

Read N scores from the user into an int array using scanf(). Write a function float average(int *scores, int n). Print the mean and count how many students scored above average.

Idea 4: Character Counter (Strings + Pointers)

Write int countChar(char *str, char target) that iterates through a string using pointer arithmetic and returns how many times target appears. Test with several calls from main().
